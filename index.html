CSS Notes

Rember: All HTML elements are boxes made up of four components: a content container, padding, border, and margin

  • Verticle margins collapse, larger margin wins
  • margin: 0 auto; will centre horizontally
  • overflow-y: scroll; will only show vertical scroll-bar
  • display: none; will remove element from the web page
  • visibility: hidden; will hide element from viewer but keep a place holder
  • By default, browsers use box-sizing: content-box;
  • Better control comes from * {box-sizing: border-box;} where the height and width will be fixed and the border thickness + padding is included INSIDE the box
